# Backend - Plan Indicativo Dashboard

Coloca aquí tu archivo Excel `PlanIndicativo.xlsx`.

El archivo debe tener las siguientes columnas (pueden estar en mayúsculas o minúsculas):
- ID
- Programa
- Descripcion (o Meta)
- T3_Plan
- T3_Ejec
- T4_Plan
- T4_Ejec
- Evaluacion (opcional, se calcula automáticamente si no está presente)
